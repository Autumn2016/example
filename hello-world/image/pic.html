1. 操作系统

1.1 Linux： 建议UBUNTU 14.04 LTS

1.2 Mac: 建议OS-X 10.10以上

2. 开发工具

2.1 IDE：JetBrains WebStorm 10

2.2 MySQL: MySQL 5.6.X (仅在本机上调试时需要配置，使用测试服务器调试时无须配置)

  • 在Linux上安装MySQL

$ sudo apt-get update # 更新源

$ sudo apt-get install mysql-server-5.6 # 设置数据库管理员密码为：“romens@2015”

  • 在Mac上安装MySQL

$ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)" # 安装brew

$ brew search mysql # 查找所有的mysql相关的可用包

$ brew install homebrew/versions/mysql56

  • 修改MySQL的root密码

$ mysql -h localhost -uroot

mysql> SET PASSWORD FOR 'root'@'localhost' = PASSWORD('romens@2015');

2.3 Redis: Redis 2.8以上 (仅在本机上调试时需要配置，使用测试服务器调试时无须配置)

  • Linux

$ apt-get update

$ apt-get install redis-server

  • Mac

$ brew install redis

2.4 安装nodejs

  • Linux

$ curl -sL https://deb.nodesource.com/setup_0.12 | sudo -E bash -

$ sudo apt-get install nodejs

  • Mac

$ brew install homebrew/versions/node012

2.5 安装其他工具所需要的npm包

$ npm install -g supervisor gulp

3. 启动scc

3.1 拉取scc源码, 用自己的github账号替换“hc-romens”

$ git clone https://hc-romens@github.com/romens-cn/scc-src.git

3.2 初始化环境变量, 其中符号"."为shell source命令，必须输入。

$ cd scc-src $ . bin/setenv.sh cd # cd 为公司测试服务器域名, 如果使用本机的MySQL和Redis服务器，请更换为127.0.0.1

3.3 初始化数据库

$ db-init.sh

3.4 安装npm依赖包

$ npm install

3.4 启动scc

$ go.sh cd

3.5 启动react开发模式（仅在开发时使用，生产环境无须执行本操作），在新的terminal中执行

$ cd quotation

$ npm run dev

4. 在浏览器访问
  • 如果是B2B模式，打开http://127.0.0.1:3300
  • 如果是A证模式（EDI模式），打开http://127.0.0.1:4000
